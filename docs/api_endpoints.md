# API Endpoints

| Método | Ruta                   | Descripción                 | Auth |
|-------:|------------------------|-----------------------------|------|
| GET    | /propiedades           | Lista propiedades           | No   |
| POST   | /propiedades           | Crea propiedad              | Sí   |
| GET    | /propiedades/:id       | Detalle de propiedad        | No   |
| POST   | /auth/login            | Login usuario               | No   |
| POST   | /auth/register         | Registro usuario            | No   |

> Respuestas y ejemplos: ver `openapi.yaml` (opcional) o ejemplos curl en este archivo.
